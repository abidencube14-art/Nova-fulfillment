# Nova AI Fulfillment 🚀

Nova AI Fulfillment is the automation engine behind Nova Select.

## Features (Planned)

- Shopify order detection
- PesaPal payment integration
- Automatic payment emails
- Order management
- Supplier matching
- Fulfillment automation

## Current Stage

Phase 1:
- Project foundation
- Shopify webhook receiver
- PesaPal payment receiver

## Setup

Install dependencies:

npm install

Run:

npm start
